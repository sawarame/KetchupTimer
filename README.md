# KetchupTimer 🍅

**KetchupTimer** is a unique Pomodoro Technique browser extension (Manifest V3) with a fun "Ketchup Bottle" concept. 
During your focus sessions, the ketchup in the bottle decreases, and during your breaks, the ketchup is refilled! This provides intuitive visual feedback on your progress.

[日本語の説明は下部にあります (Japanese description below)](#japanese)

## Features (English)
- **Pomodoro Timer:**
  - Focus phase: 25 minutes (default)
  - Short break: 5 minutes (default)
  - Long break: 15 minutes (default)
  - Cycle: Automatically transitions to a long break after a specified number of focus cycles (default 4).
- **Interactive UI (Popup):**
  - Displays current phase, remaining time, and cycle count.
  - Easy-to-use Start, Pause, and Reset controls.
  - Customizable timer durations and cycle counts via settings.
- **Smart Notifications:**
  - System notifications alert you when a phase ends.
  - Click "OK" on the notification to seamlessly transition to the next phase.
- **Dynamic Extension Icon:**
  - The extension icon in your browser toolbar updates dynamically based on the remaining time, visually showing the ketchup level dropping and refilling.

## 概要 (Japanese)
**KetchupTimer（ケチャップタイマー）** は、ポモドーロ・テクニックを用いたブラウザ拡張機能（Manifest V3対応）です。
最大のコンセプトは「ケチャップボトル」であり、作業中（Focus）はケチャップが減っていき、休憩中（Break）はケチャップが補充されていくというユニークな視覚的フィードバックを提供します。

## 機能
- **タイマー機能:**
  - 集中（Focus）フェーズ：デフォルト25分
  - 短い休憩（Break）フェーズ：デフォルト5分
  - 長い休憩（Long Break）フェーズ：デフォルト15分
  - サイクル：指定回数（デフォルト4回）の集中フェーズを終えると、長い休憩に移行します。
- **UI (Popup):**
  - 現在のフェーズ、残り時間、サイクル数を表示します。
  - Start, Pause, Reset コントロールボタンを搭載。
  - 各フェーズの時間設定とサイクル数のカスタマイズが可能です。
- **通知 (Notifications):**
  - フェーズ終了時にChromeのシステム通知でお知らせします。
  - 通知内のボタン（OK）をクリックすることで、次のフェーズへシームレスに移行できます。
- **動的アイコン演出:**
  - タイマーの残り時間に応じて、ブラウザツールバーのアイコンが動的に更新されます。
  - ケチャップの液面が変化するアニメーションで残り時間を直感的に把握できます。
