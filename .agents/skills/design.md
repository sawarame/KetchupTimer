# Design & CSS Skill

## ケチャップ・テーマ
- **ベースカラー:** `#ff4d4d` (ケチャップレッド)
- **スタイル:** 丸みを帯びたポップなデザイン、フラットデザイン。
- **一貫性:** ケチャップボトルをモチーフにした視覚的フィードバック（Focusで減り、Breakで増える）。

## CSS 原則
- **Vanilla CSS:** 外部フレームワークを使用せず、標準的な CSS3 で実装する。
- **レスポンシブ:** Popup のサイズ制限（最大 800x600px）を考慮し、コンパクトなレイアウトを保つ。
- **変数管理:** カラーコードやフォントサイズは CSS 変数 (`--primary-color` 等) で定義し、保守性を高める。

## UX 原則
- **最小操作:** 通知からのワンクリック遷移など、ユーザーの手間を減らす。
- **フィードバック:** タイマーの状態をアイコンの液面変化で直感的に伝える。
